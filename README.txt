
# Док Куриленко — Telegram-бот

## Что внутри:
- Шаблон консультативного заключения
- Шаблон УЗИ для беременных
- Webhook-версия для Render
- Генерация PDF со шрифтом DejaVuSans

## Инструкция по запуску на Render:
1. Создайте новый Web Service.
2. Укажите:
   - `BOT_TOKEN` — токен вашего Telegram-бота
   - `WEBHOOK_URL` — https://<имя-проекта>.onrender.com/webhook
3. Готово!
